# 01 — Alocação Dinâmica de Memória

Exercícios sobre alocação dinâmica de memória em C usando `malloc`, `calloc`, `realloc` e `free`.

## Exercícios

| Arquivo | Descrição |
|---|---|
| `ex1.c` | Alocação com `calloc` — inserção e consulta em vetor dinâmico |
| `ex2.c` | Manipulação de memória alocada dinamicamente |
| `ex4.c` | Exercício com ponteiros e alocação |
| `ex5.c` | Alocação dinâmica com operações variadas |
| `ex6.c` | Alocação e liberação de memória |
| `ex7.c` | Exercício avançado com alocação dinâmica |
| `ex8.c` | Manipulação de dados em memória dinâmica |
| `ex9.c` | Operações com memória dinâmica |
| `ex10.c` | Exercício final de alocação |

## Como compilar

```bash
gcc ex1.c -o ex1
./ex1
```
